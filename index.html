How does it work?

Quite simply actually.

  1. Create your language selector links/buttons
  2. Set the id on those selectors to "choose-[language]" where [language] is english or espanol or any language name in lowercase: id="choose-english"
  3. Add your content to the html in each language (remember, many images and other elements can be shared by all languages)
  4. Give each element a class that is the same as the [language] you set for the selector: class="english" or class="espanol"
  5. Adjust the variable settings the the language.jquery.js file as directed
